# HAL Example Missions

Concrete, step-by-step mission recipes. Each one tells you exactly what to
place in Eden, how to sync it, and what to watch for when previewing.

## Available examples

| Example | Complexity | Time to build | What it demonstrates |
|---|---|---|---|
| [Basic 2-Side Battle](basic-battle.md) | Beginner | 5 min | One HQ per side, autonomous combat, fire support basics |
| [Three-HQ Combined Arms](three-hq-with-arty.md) | Intermediate | 20 min | Multiple HQs with different roles, personality tuning, squad modifiers |
| [Player as HAL Commander](player-commander.md) | Intermediate | 10 min | Hybrid mode — player commands via action menu, HAL dispatches |

## Progression recommendation

1. Build **Basic 2-Side Battle** first. Five minutes, works on VR, gives you
   a sense of HAL's baseline behavior.
2. Build **Player Commander**. This teaches you the action menu and the
   support-request flow.
3. Build **Three-HQ Combined Arms**. This is where HAL's multi-HQ coordination
   and per-HQ tuning become interesting.

Each example is self-contained — you don't need to do them in order, but
they're listed roughly from simplest to most complex.
